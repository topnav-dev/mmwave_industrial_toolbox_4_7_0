Version 2.2.0.0 ============= Overview ======== This lab demonstrates the use of TI mmWave sensors for Area Scanner applications. The unique range, velocity, and angle data from mmWave sensors can enable the detection of objects or people as they enter or exit zones of interest. * SDK Version: 3.5.0.4 * Supported EVM platforms: * IWR6843ISK (ES2.0) * IWR6843ISK-ODS (ES2.0) * IWR6843AOPEVM (Rev F) New and Updated Features ======================== * SDK 3.5 Update Known Issues & Work Arounds ============ * An unresolved UART issue can result in missing bytes in packets. The work around used in this release is to check and skip parsing of frame with missing bytes in gui. Limitations ========================= * The minimum range of the static object detection is currently limited by the occurence of false detections. --------------------------------- Release Notes Archive ========================= ## Version 2.0.0.0 New and Updated Features: * IWR6843ISK-ODS (ES2.0) support. Known Issues & Work Arounds: * An unresolved UART issue can result in missing bytes in packets. The work around used in this release is to check and skip over parses with missing bytes. * There can be false static object detections in the range closest to the EVM. The current work around is to set the start range bin for static detection to the 17th range bin. This limits the minimum detection region for static objects to ~1m. Limitations: * The minimum range of the static object detection is currently limited by the occurence of false detections.
