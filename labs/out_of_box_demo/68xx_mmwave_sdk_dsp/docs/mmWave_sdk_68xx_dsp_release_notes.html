Version 3.5.0 ============= Overview ======== This is the mmWave SDK out-of-box demo lab for the TI IWR68xx EVM. The mmWave Demo is used with an online GUI to show the visualization of processed radar data. The demo source code is pulled from the mmWave SDK when the project is imported into CCS. The GUI is configurable and is able to display several different plots. **NOTE**: This version of the SDK out-of-box demo is for IWR68xx EVM and uses both the on-chip Hardware Accelerator (HWA) and on-chip c674x DSP.
