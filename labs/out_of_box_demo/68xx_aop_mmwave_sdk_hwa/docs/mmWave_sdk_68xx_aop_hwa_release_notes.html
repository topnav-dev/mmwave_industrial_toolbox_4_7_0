Version 3.5.0 ============= Overview ======== This is the mmWave SDK out-of-box demo lab for the TI IWR6843AOP EVM. The mmWave Demo is used with an online GUI to show the visualization of processed radar data. The demo source code is pulled from the mmWave SDK when the project is imported into CCS. The GUI is configurable and is able to display several different plots. Features ======== The following features are supported: * Supports mmWave SDK 3.5.x.x * Supports IWR6843AOPEVM Rev F (or later). Note: Not compatible with prior versions of IWR6843AOPEVM.
