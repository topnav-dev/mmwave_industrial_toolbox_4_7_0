Overview ======== This lab demonstrates the use of TI mmWave sensors to track and count people using the IWR6843ISK-ODS or IWR6843AOPEVM sensor module mounted directly above the area of interest (i.e. overhead mounting). Detection and tracking algorithms run onboard the IWR6843 mmWave sensor and are used to localize people and track their movements. The lab provides full source code and CCS projects for building the binaries from source. Features ======== The following features are supported: * Supports IWR6843ISK-ODS antenna module (ES2.0) * Supports IWR6843AOP antenna module (ES2.0, Rev F or later). * Supports a new FCC compliant low bandwidth mode (compile time selectable) * GUI displays point cloud and tracking information * Supports mmWave SDK 3.5 New and Updated Features ======================== The following features are new or updated in this release: **Documentation update:** The following new documents covering People Counting demo software implementation, Detection and Tracker layer parameter tuning, and Tracker algorithm and implementation details have been released in labs\people_counting\docs directory. 1. 3D_people_counting_demo_implementation_guide.pdf 2. 3D_people_counting_detection_layer_tuning_guide.pdf 3. 3D_people_counting_tracker_layer_tuning_guide.pdf 4. Tracking_radar_targets_with_multiple_reflection_points.pdf Resolved Incident Reports ========================= The following are Incident Reports resolved in this release: * N/A Known Issues ============ The following are Known Issues in this release: * Device must be restarted for reconfiguration. * **Delayed track allocation in NE and NW quadrants (positive Z-axis) under certain conditions**: There may be a delay in track allocation in the NE and NW quadrants for targets walking towards the center position. The tracking performance is not affected while walking away from the center position in the same quadrants. * **Mirroring in EVM elevation axis (Z-axis)**: Objects/People at the edge of elevation FoV (around +/- 60 degrees) may result in ghost objects in the opposite direction (i.e. Mirroring) * **Height distortion in EVM elevation axis (Z-ais)**: The height of the objects/people at the edge of elevation FoV (around +/- 60 degrees) may appear to be higher than actual in the point cloud display.
