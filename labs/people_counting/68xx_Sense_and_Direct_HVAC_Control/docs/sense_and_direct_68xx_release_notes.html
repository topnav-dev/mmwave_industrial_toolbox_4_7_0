Sense and Direct - HVAC Control with IWR68xx 1.2.0 ============= Overview ======== This software package implementing false detection mitigation and static detection for people counting and tracking applications such as HVAC control. The demo minimizes false positives through classification of indoor objects such as fans, curtains, blinds, moving plants vs humans. Features ======== The following features are supported: * Group tracker for counting and tracking people * Detection of Static People * Classifier for more intelligent people counting and tracking by enabling robustness to sources of false motion such as fans. * Supports mmWave SDK 3.5 * Supports IWR6843 ES2.0 devices.
