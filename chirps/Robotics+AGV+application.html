Robotics AGV application

Introduction

This use case provides the mmWave radar sensor configuration for Robotics / AGV (Autonomous Ground Vehicle) applications. In Robotics, mmWave radar can be used for multiple applications such as Ground Speed Sensing, Edge Detection, Avoiding collisions with objects or people etc. Typically, the maximum range and velocity requirements for such applications are fairly moderate however the system needs to be able to detect very small objects (low RCS) and also have good range and velocity resolutions.

Testing and Results

This chirp configuration needs a dedicated processing chain to work and it cannot work with OOB Demo processing chain. However, this chirp configuration has been tested to confirm that it correctly configures the Radar Front-End for the specified sensing configuration i.e. it doesn't generate any mmWaveLink API errors. This is done by loading the chirp configuration in the OOB Demo Configuration GUI (using the OOB Demo Configuration File provided in Step 3).
